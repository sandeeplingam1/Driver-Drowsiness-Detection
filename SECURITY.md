Security Policy
===============

The Driver Drowsiness Detection System is committed to ensuring the safety and security of its users.

Supported Versions
------------------

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0.0 | :x:                |

Reporting a Vulnerability
-------------------------

Please do not report security vulnerabilities through public GitHub issues. Instead, please follow the process outlined below:

1. Send an email to the maintainer with a detailed description of the vulnerability.
2. Include steps to reproduce and potential impact.
3. Allow up to 48 hours for an initial response.
4. We coordinate a fix and a public disclosure timeline.

Responsible disclosure ensures the safety of everyone using this system.
